¿Wie werde ich Fon Mitglied?

Die Teilnahme am Fon WLAN Netzwerk ist einfach:

  • Kaufen Sie einen Fon Router oder Prüfen Sie ob Ihr Internetanbieter über eine Partnerschaft mit Fon verfügt und die Fon-Funktion auf Ihrem Router aktiviert ist (falls das der Fall ist, brauchen Sie sich nur für den Dienst anzumelden).
  • Schließen Sie die Fonera an Ihrem bestehenden Breitband Internetanschluss an.
  • Registrieren Sie Ihrer Fonera und genießen Sie kostenloses WLAN Roaming im gesamten Fon Netzwerk.

Wenn Sie Fon anschließen, teilen Sie einen kleinen Teil der Breitbandverbindung und erhalten im Gegenzug kostenlosen Zugriff auf Millionen von anderen Fon Spots.

Falls Sie keine Fonera an Ihrem Breitband anschliessen können, können Sie trotzdem Zugang zum Fon WLAN genießen indem Sie Fon Zugangspässe überall dort kaufen wo immer Sie das FON_FREE_INTERNET Signal sehen. Erstellen Sie einfach ein Fon Konto und kaufen Sie einen Zugangspass via Kreditkarte oder SMS.
